# Fathima Asra Suhair - Portfolio

A modern, professional portfolio website built with React.js, featuring smooth animations and an attractive dark theme.

## 🌟 Features

- **Modern Design**: Dark theme with purple/blue gradient accents
- **Smooth Animations**: Powered by Framer Motion for eye-catching transitions
- **Fully Responsive**: Looks great on all devices
- **Interactive Components**: Hover effects, scroll animations, and more
- **Professional Sections**:
  - Hero section with social links
  - About Me with statistics
  - Technical Skills showcase
  - Featured Projects with detailed information
  - Contact form

## 🚀 Technologies Used

- React.js
- CSS3
- Framer Motion (animations)
- React Icons
- React Intersection Observer

## 📦 Installation

1. Clone or download this repository

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser

## 🛠️ Build for Production

To create a production build:

```bash
npm run build
```

The optimized files will be in the `build` folder, ready to be deployed.

## 🎨 Customization

You can easily customize the portfolio by editing:

- **Personal Information**: Update text content in each component
- **Colors**: Modify the gradient colors in CSS files
- **Projects**: Edit the projects array in `Projects.js`
- **Skills**: Update the skillCategories array in `Skills.js`
- **Contact Info**: Change email and social links in `Hero.js`, `Contact.js`, and `Footer.js`

## 📱 Sections

1. **Hero**: Eye-catching introduction with animated text and social links
2. **About**: Personal bio with statistics
3. **Skills**: Organized technical skills with icons
4. **Projects**: Showcase of major projects with descriptions
5. **Contact**: Contact form and information
6. **Footer**: Quick links and additional social connections

## 🌐 Deployment

You can deploy this portfolio to various platforms:

- **Netlify**: Drag and drop the build folder
- **Vercel**: Connect your GitHub repository
- **GitHub Pages**: Use `npm run build` and deploy the build folder
- **AWS S3**: Upload build files to an S3 bucket
